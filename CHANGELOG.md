## [0.1.0] - 2026-02-10

### Added
- Initial release
- Private chat and group chat support (@mention trigger)
- Image message support (auto download + AI analysis)
- Context reset (`/reset`, `/重置`)
- User/group allowlist
- Auto message deduplication
- WebSocket auto-reconnect
- Optional HTTP endpoint for proactive messaging
- OpenClaw plugin lifecycle management (auto start/stop)
